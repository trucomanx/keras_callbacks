# keras_callbacks

A set of new callbacks for Keras


## Install from PYPI

The homepage in pipy is https://pypi.org/project/keras_callbacks/

```bash
pip install --upgrade keras_callbacks
```

## Install from source
Installing `keras_callbacks` 

```bash
git clone https://github.com/trucomanx/keras_callbacks.git
cd keras_callbacks
pip install -r requirements.txt
cd src
python -m build
pip install dist/keras_callbacks-*.tar.gz
```

## Uninstall

```bash
pip uninstall keras_callbacks
```
